# sorting-proyect
Implementacion en C de los algoritmos de ordenamiento HeapSort y RadixSort, incluyendo analisis de complejidad, pruebas experimentales y comparacion de rendimineto bajo distintos escenarios de entrada bajos ciertos criterios


make          # Compila todo
make clean    # Limpia archivos generados
make test     # Compila y ejecuta test de listas
make test_heap # Compila y ejecuta test de heapSort
make debug    # Compila en modo depuración
